# 115转存助手 (115 Auto Save)

这是一个用于115网盘分享页面的油猴脚本，自动点击提取码提交、一键转存和确认转存按钮，简化转存流程。

## 功能特性

- ✅ **自动提交提取码**：自动点击提取码提交按钮
- ✅ **一键转存**：自动点击一键转存按钮
- ✅ **确认转存**：自动点击立即转存确认按钮
- ✅ **智能等待**：使用时间间隔确保操作顺序
- ✅ **错误处理**：完善的元素查找和错误提示

## 安装方法

### 1. 安装油猴脚本管理器

首先需要安装浏览器扩展：
- **Tampermonkey**（推荐）：[Chrome商店](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
- **Violentmonkey**：[官网下载](https://violentmonkey.github.io/)

### 2. 安装脚本

有以下几种安装方式：

#### 方式一：直接安装（推荐）
1. 点击脚本文件链接
2. 油猴脚本管理器会自动提示安装
3. 点击"安装"按钮即可

#### 方式二：手动安装
1. 复制脚本代码
2. 打开油猴脚本管理器
3. 点击"添加新脚本"
4. 粘贴代码并保存

## 使用方法

### 基本操作

1. **打开115分享链接**：访问格式为 `https://115cdn.com/s/...` 的分享页面
2. **等待脚本自动执行**：脚本会在页面加载完成后自动运行
3. **观察执行过程**：在浏览器控制台查看执行日志

### 执行流程

脚本按照以下顺序自动执行：

1. **等待0.5秒** → **点击提取码提交按钮**
2. **等待0.5秒** → **点击一键转存按钮**
3. **等待0.5秒** → **点击立即转存确认按钮**

### 支持的页面类型

- 115网盘分享页面 (`https://115cdn.com/s/*`)
- 需要输入提取码的分享页面

## 技术特性

- **兼容性**：支持Chrome、Firefox、Edge等现代浏览器
- **安全性**：仅访问115网盘相关域名
- **性能优化**：使用async/await确保操作顺序

## 常见问题

### Q: 脚本不生效怎么办？
A: 检查以下事项：
- 油猴脚本管理器是否已启用
- 是否在正确的115分享页面 (`115cdn.com/s/...`)
- 浏览器控制台是否有错误信息
- 页面是否完全加载完成

### Q: 脚本执行太快或太慢怎么办？
A: 可以修改脚本中的等待时间（sleep函数参数）来调整执行速度

### Q: 如何查看执行日志？
A: 打开浏览器开发者工具（F12），在Console标签页查看执行日志

### Q: 如何更新脚本？
A: 油猴脚本管理器会自动检查更新，或手动重新安装最新版本。
